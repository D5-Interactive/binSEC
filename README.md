- **Prism launcher:** [ https://prismlauncher.org/ ]

- **To play,** download the CLIENT **.mrpack** file, install PRISM MODDED LAUNCHER; import the .mrpack - input the IP into the address section when "Adding server" in the multiplayer menu.
6-8GB ram dedicated minimum.


## Performance Recommendations
### Client

RAM: 6-8GB allocated
Shaders: Complementary or BSL recommended
Render Distance: 12-16 chunks

### Server

RAM: 8-12GB for 10-20 players
CPU: Modern multi-core recommended
Pregeneration: 5000 block radius minimum
Backups: Use scheduled backups (not included)

**Minecraft Version:** 1.21.1  
**Loader:** NeoForge 21.1.219
